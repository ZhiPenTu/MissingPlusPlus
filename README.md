# 心安日记

> 写给**焦虑型依恋**的人：一个住在 macOS 菜单栏的"浪来时"伙伴。

`心安日记` 是一个常驻菜单栏的 macOS 应用，专门为**焦虑型依恋人格**（anxious attachment）设计。
它不教你"不要想"，而是陪你看清自己的想念 — 看清浪什么时候来、什么时候走、走了多少次、走过之后你真的平复过。

## 它是为谁做的

如果你有过这样的时刻：

- 消息发出去 5 分钟没回，已经在脑子里排练了 17 种"TA 不爱了"的剧本
- 一个人待着就开始想 TA，循环刷聊天记录，越刷越难受
- 浪来时知道自己不该发消息，但就是接不住自己
- 浪走了之后又后悔"刚才又小题大做了"
- 反复问自己"我是不是太粘人 / 太敏感 / 太作"

那你就是这款应用的目标用户。

它不评判情绪，不催促"振作起来"，只是把每一次想念记下来，让你慢慢看见自己的 pattern — 看见浪的形状，浪就会没那么可怕。

## 它能做什么

### 一、一键记下想念
菜单栏点一下，3 秒记一笔：想念谁、什么心情、多强烈。
不需要写长文，不需要"为什么想"，先把那一刻的浪落下来再说。

### 二、看清 Trigger 模式
每次想念可以打上 trigger 标签 — "💬 TA 没及时回""🔇 已读不回""⚡️ 刚吵完架""🏠 独处"……
积累到一周，回看会发现：原来 80% 的浪都来自那 3 个 trigger，剩下 20% 才是"莫名其妙的想念"。
看见 pattern，是焦虑型依恋康复的起点。

### 三、标记"浪过去了"
浪来时不用做任何事 — 但浪走了之后，点一下"平复了"。
应用会帮你算：过去 30 天你的平复率是多少，平复一次平均需要多久。
这是焦虑型最缺的"累积证据"：你不是每次都会失控的。

### 四、浪来时接住你（Self-Soothing）
浪来时不想动脑？三个 body 层工具点开就引导：

- **5-4-3-2-1 grounding** — 看 5 样 / 触 4 样 / 听 3 样 / 闻 2 样 / 尝 1 样，30 秒把自己拉回当下
- **自我同情 break** — Kristin Neff 风格的 17 句 curated 短语，一句一句翻给自己听
- **Cooldown 活动** — 6 条预定义 DBT 活动（喝杯水、出门走 5 分钟、深呼吸 10 次…），也可以加自己的

### 五、现实检验（DBT "Check the Facts"）
强烈想念时，应用会温柔地问你一句：
> "支持你'TA 不爱你了'这个念头的证据是什么？反对的证据呢？"

不是要你否定自己的感受，是 DBT 里的标准练习：把 catastrophize 的念头摆到桌面上，和事实对照。
很多次写完这一栏，浪已经过了一半。

### 六、致 TA 的话（不发出去的版本）
AI 帮你写一封"想对 TA 说、但不会发出去"的信。
可以换一封，可以复制。
写完就关 — 焦虑型最怕的不是不会表达，是"说了会后悔"。这封信只给你自己看。

### 七、统计趋势
过去 30 天：
- 浪都平复了吗？平复率多少？
- 哪种心情出现得最多？
- 哪些 trigger 是你的高频雷区？
- 一天中几点最容易想 TA？

数据存在本地，不上传。

## 隐私

- **完全离线**：除可选的 AI 增强外，应用不发任何网络请求
- **本地存储**：数据存在 `~/Library/Application Support/MissingPlusPlus/missings.json`，App Sandbox 容器内
- **可选同步**：可以导出 JSON 备份 / 导入到新设备；可以走 iCloud KV 同步
- **可选 AI**：AI endpoint 和 API key 是你在 Settings 里自己配的，应用不内置任何服务

## 系统要求

- macOS 13+（Debug 构建）/ macOS 26+（当前 Release DMG，因 Xcode 26 不再 embed Swift stdlib）
- Apple Silicon 或 Intel Mac

## 下载 / 安装

Release DMG 在 [dist/MissingPlusPlus-1.0.dmg](dist/)。双击挂载，把 `心安日记.app` 拖进 Applications 文件夹。

首次启动 macOS 可能提示"无法打开，因为它来自身份不明的开发者" — 系统设置 → 隐私与安全性 → 仍要打开。

## 它不是什么

- **不是治疗工具**：它不替代心理咨询师 / 治疗师，也不做诊断
- **不是社交 App**：不连 TA、不通知 TA、不替你发消息
- **不是 journal app 的通用版**：它只做"想念"这一件事，并把这个场景做透
- **不是数据收集器**：不上传、不分析、不卖你的情绪数据

## 一个不成熟的小想法

如果你用了一段时间，发现自己：
- 浪来得没那么密了
- 浪来了知道"这只是浪"
- 浪走了真的会平复，而不是带进下一天

那这个应用就完成它的工作了。

如果想得 TA 的频率没变，但**对浪的恐惧少了一点** — 也算赢。
